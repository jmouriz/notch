# Contribuir a Notch

Gracias por ayudar a mejorar Notch.

## Preparación

1. Usá macOS 14 o posterior con Xcode y Swift 6.
2. Creá una rama desde `main`.
3. Implementá cambios acotados y agregá pruebas cuando corresponda.
4. Ejecutá `swift test` antes de abrir un pull request.

## Pull requests

- Explicá el problema y el resultado esperado.
- Incluí capturas cuando el cambio afecte la interfaz.
- No incluyas contenido descargado, cachés, aplicaciones ni imágenes DMG.
- Conservá los avisos de licencia de las herramientas incluidas.

## Reportes

Al informar un error, indicá la versión de macOS, el modelo de Mac, los pasos
para reproducirlo y cualquier mensaje mostrado por Notch.
